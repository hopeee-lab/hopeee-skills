## Skills

### gpt-image-prompt-helper

GPT-Image-2 Prompt 助手。当用户想生成图片、寻找 prompt 模板时触发，从以下两个社区 prompt 库中检索最匹配的案例，并根据用户需求改写返回。

**数据源：**
- [EvoLink awesome-gpt-image-2-prompts](https://github.com/EvoLinkAI/awesome-gpt-image-2-prompts)
- [YouMind awesome-gpt-image-2](https://github.com/YouMind-OpenLab/awesome-gpt-image-2)（含 Web Gallery：youmind.com/gpt-image-2-prompts）

**触发词：** 生成图片、写 prompt、帮我画、prompt 模板、图生文
